=========== CorpusForge 开发进度 ===========

【第三轮更新 — 2026-05-08 上线差距优化】

✅ ErrorBoundary + 错误页面体系
  - ErrorBoundary.tsx：全局错误边界组件（捕获渲染错误、显示重试/刷新按钮、开发环境展示堆栈）
  - NotFoundPage.tsx：404 页面（返回上一页/回到首页）

✅ API 服务层搭建
  - api/client.ts：axios 实例封装（请求/响应拦截器、统一错误处理、401 自动跳转登录）
  - api/corpus.ts：语料 + 语料库 API（CRUD、批量操作、文件上传、导出）
  - api/annotation.ts：标注任务 + 审核 API（CRUD、生命周期管理、审核工作流）
  - api/admin.ts：认证 + 用户管理 + 搜索 + 统计 + 系统 API
  - api/index.ts：统一导出

✅ 环境变量配置体系
  - .env：通用配置（VITE_APP_TITLE、VITE_APP_VERSION）
  - .env.development：开发环境（API_BASE_URL、MOCK_ENABLED）
  - .env.production：生产环境配置
  - vite.config.ts 增强：路径别名(@)、开发代理、chunk 分包优化

✅ 标注工作区核心功能（AnnotationWorkspace.tsx）
  - NER 实体标注：文本选中高亮标注（选标签 → 拖选文本 → 自动标记）
  - 分类/情感/意图标注：标签选择式标注（单选/多选）
  - 标注导航：上一条/下一条、进度条、已完成/跳过计数
  - 标注历史：撤销/重做（最多50步）
  - 全屏模式：标注沉浸式体验
  - 已标注实体列表：可视化展示、支持删除
  - 标记完成/跳过/提交所有标注
  - AnnotationWorkspacePage.tsx：路由级包装组件
  - AnnotationDetail.tsx 增加"开始标注"入口按钮
  - 路由新增 /annotation/:taskId/workspace

✅ 前端权限控制系统
  - PermissionGuard.tsx：权限守卫组件（角色检查、权限检查、高阶组件 withPermission）
  - usePermission Hook：hasRole、hasAnyRole、hasPermission、isAdmin、isSuperAdmin
  - Layout.tsx 菜单权限过滤：根据用户角色动态显示菜单项

✅ 全局样式体系完善
