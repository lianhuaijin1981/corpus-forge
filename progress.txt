=========== CorpusForge 开发进度 ===========

【最近更新 — 2026-05-08 第二轮】

✅ 语料检索模块深度增强（SearchPage.tsx 完整重写）
  - HighlightText 子组件：搜索结果关键词高亮显示
  - CorpusPreviewModal 子组件：语料详情预览弹窗（Descriptions + 内容预览 + 高亮）
  - exportToCSV 函数：搜索结果导出 CSV（BOM UTF-8 编码，Excel 兼容）
  - 收藏功能：通过 localStorage 持久化（key: corpusforge-search-favorites）
  - 搜索历史：通过 localStorage 持久化（key: corpusforge-search-history），支持清空
  - 高级筛选新增 RangePicker 日期范围筛选

  - 新增语料库快速检索区（按语料库过滤语料）

✅ 统计分析模块深度增强（StatsPage.tsx 完整重写）
  - 接入 reviewStore 计算审核通过率、待处理数量等审核统计
  - growthTrendOption：近12个月增长趋势双轴图（柱状图=新增语料，折线=累计总量）
  - taskProgressOption：标注任务完成率横向柱图 Top 8
  - 三个进度条卡片：标注完成率 / 语料通过率 / 审核通过率
  - 核心指标卡片增加子状态 Tag（今日新增、本周新增、待审核等）
  - exportStatsReport：导出 JSON 格式统计报告

✅ 系统管理模块深度增强（AdminPage.tsx 完整重写）
  - 操作日志系统：appendLog + loadLogs + saveLogs（key: corpusforge-operation-logs，最多200条）
  - 系统配置：loadConfig + saveConfig（key: corpusforge-sys-config），支持保存/恢复默认
  - BatchRoleModal 子组件：批量分配角色弹窗
  - 用户管理Tab顶部统计卡片行（总数/活跃/待审核/已禁用 + 角色分布Badge）
  - Table 增加 rowSelection 复选框，支持批量禁用/启用/分配角色
  - 系统配置Tab改为 Switch + Input 可编辑表单
  - 操作日志Tab支持搜索过滤，日志含彩色状态 Tag
  - 全部确认操作使用 Popconfirm 替代 Modal.confirm

✅ 审核中心模块完善（ReviewList.tsx 重写 + reviewStore.ts 新建）
  - reviewStore.ts：审核任务状态管理（Mock 数据 + Zustand + persist）
  - ReviewList.tsx 完整重写：统计卡片、多维筛选、批量通过/驳回、审核详情弹窗

✅ 其他修复与优化

  - CorpusForm.tsx：语料表单优化
  - TypeScript 编译零错误（tsc --noEmit 全部通过）
